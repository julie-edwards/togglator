Togglator

Thanks for trying togglator! Togglator is an easy to use jQuery plugin that creates toggling content panels. Try out some of the settings below, and visit me at julie-edwards.com to learn more.

Default settings!

hideInitial : true,
animate : true,
animateSpeed : 400,
arrow : true,
rotateArrow : true,
animateArrow : true,
defaultTitle : "View Content"

rotateArrow : false,
animateSpeed : 200
